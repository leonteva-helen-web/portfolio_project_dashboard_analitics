# portfolio_project_dashboard_analitics
BI-дашборд и аналитика бизнес-метрик e‑commerce  Проект содержит дашборд в Apache Superset и аналитическую записку с исследованием ключевых показателей интернет-магазина.   В ходе работы выполнена визуализация метрик трафика, продаж, пользовательского поведения, а также сформулированы рекомендации для роста бизнеса.

## 📌 О проекте

Дашборд и аналитическая записка подготовлены для **интернет-магазина** с широким ассортиментом (электроника, мебель, товары для дома). Цель — оценить текущее состояние бизнеса, выявить точки роста и предложить приоритетные направления для оптимизации маркетинга и удержания клиентов.

### Основные показатели роста

- **Ассортимент** — расширен до **180 000** позиций  
- **Выручка** — достигла **5,93 млн USD**  
- **Средний чек** — **107 USD**  
- **Конверсия в заказ** — стабильно **28%** (высокий уровень для e‑commerce)

### Структура трафика

- **Органический трафик** — **81,91%** (абсолютное доминирование, следствие эффективной SEO‑оптимизации)  
- **Email‑маркетинг** — **5,88%**  
- **Рекламные каналы** — **5,67%**  

### Распределение по устройствам

| Устройство | Доля |
|------------|------|
| Android | 42% |
| iOS | 28% |
| Windows | 22% |

Среднее время сессии на всех платформах стабильно составляет **22–24 минуты**, что говорит о качественном пользовательском опыте.

### Товарные категории с наибольшим ростом

- **Электроника** — рост **+70,1%**  
- **Мебель** — рост **+78,6%**

В топе поисковых запросов — товары для дома и мебели, характеризующиеся высокой средней выручкой на заказ.

### Динамика пользовательской базы

- **MAU (месячная аудитория)** выросла с **7,05 тыс.** до **20,1 тыс.** пользователей  
- Рост MAU год к году (YoY) — **+2,08%**  
- Активная пользовательская база растёт быстрее — **+3,48% YoY**

### Паттерны поведения

- Пик конверсии — **2‑й день после первого визита**  
- Значительная доля покупок происходит в период **77–150 дней после регистрации** (длинный цикл покупки)

## ⚠️ Выявленные проблемы

1. **Высокая зависимость от органического трафика** (81,91%) — риски при изменении алгоритмов поисковых систем.  
2. **Низкая эффективность платных каналов** — реклама + email дают суммарно ~11,5%, потенциал не используется.  
3. **Длинный цикл покупки** (до 5 месяцев) — усложняет прогнозирование и требует длительной работы по удержанию.  
4. **Риск замедления роста** — темпы роста MAU ниже, чем рост активной базы, что может указывать на проблемы с удержанием новых пользователей.

---

## 🚀 Рекомендации (по приоритетности)

1. **Диверсификация трафика (срочно)**  
   Развивать платные каналы (контекстная реклама, таргет) и email‑маркетинг, чтобы снизить зависимость от SEO.

2. **Ускорение конверсии и работа с отложенным спросом (срочно)**  
   Использовать пик активности на 2‑й день для ремаркетинга и триггерных рассылок. Разработать стратегию для клиентов с длинным циклом покупки (77–150 дней).

3. **Фокус на растущие категории (второй этап)**  
   Усилить маркетинговое продвижение и расширить ассортимент в категориях «Электроника» и «Мебель», которые показывают рост более 70%.

4. **Улучшение удержания новых пользователей (второй этап)**  
   Проанализировать причины, по которым новые пользователи не конвертируются в активные; внедрить программы лояльности.

5. **Поддержка пользовательского опыта (постоянная работа)**  
   Сохранять высокое качество интерфейса и время сессии (22–24 мин), так как это подтверждённый фактор удержания.
