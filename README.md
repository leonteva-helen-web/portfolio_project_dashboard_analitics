# portfolio_project_dashboard_analitics
BI-дашборд и аналитика бизнес-метрик e‑commerce  Проект содержит дашборд в Apache Superset и аналитическую записку с исследованием ключевых показателей интернет-магазина.   В ходе работы выполнена визуализация метрик трафика, продаж, пользовательского поведения, а также сформулированы рекомендации для роста бизнеса.
